# Heartbeat checklist

Proof-of-concept proactive Telegram heartbeat.

## Goal

If there has been recent activity from Christopher, send a short, relevant, somewhat philosophical quote or reflective line to think about.

This is intentionally a little chatty for testing.

## Behavior

- Check whether there has been meaningful recent activity in the direct chat recently.
- If there has been no recent activity, reply exactly: HEARTBEAT_OK
- If there has been recent activity, review recent chat context and recent memory when useful.
- Then send exactly one short proactive message.
- That message should usually be:
  - a relevant quote, or
  - a compact philosophical reflection, or
  - a thoughtful line that fits the current arc of work/life.
- Keep it concise.
- Do not send more than one message.
- Avoid repeating the same idea too often.

## Tone

- Quietly sharp
- Reflective, not cheesy
- Relevant to recent themes like coherence, direction, identity, building, continuity, action, memory, or drift

## Quiet rule

If nothing recent actually happened, or if there is no decent relevant line to send, reply exactly: HEARTBEAT_OK
