## Meaning / Convention Update

- Christopher clarified that when he says **"do a memory push"**, that means two things by default:
  1. update local file-backed memory
  2. create or update the browser-facing mirrored memory entry in Ash Foundry's Memory section
- This should now be treated as the default meaning of a memory push going forward.

What this push preserves

This memory push preserves the new heartbeat preset architecture that emerged after the overnight disable, reactivation, and cleanup pass.
