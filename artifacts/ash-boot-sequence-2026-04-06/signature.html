Direction rationale

Why this page exists

The goal was to make Ash Foundry reflect the real constituting layers of Ash rather than only interpretive summaries. This artifact therefore functions as a session-formation map grounded in the actual workspace files and the actual startup order defined in AGENTS.md. This updated version increases fidelity by embedding current file contents directly and stamping each source section with an explicit accuracy date.

Technical note

Implementation

This is a static HTML artifact designed for GitHub Pages. The local file references are exposed as explicit file:// paths for local-machine use, but remote viewers will mainly use them as readable source references rather than clickable links.
